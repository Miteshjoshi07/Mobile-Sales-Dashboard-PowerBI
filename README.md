# 📊 Mobile Sales Dashboard | Power BI

## Project Overview

This project presents an interactive Mobile Sales Dashboard developed in Microsoft Power BI.

The dashboard helps analyze mobile sales performance across different cities, brands, payment methods, customer ratings, and time periods.

---

## Dashboard Features

- Total Sales KPI
- Total Quantity Sold
- Total Transactions
- Average Price
- Monthly Sales Trend
- Sales by Mobile Model
- Sales by City (Map)
- Payment Method Analysis
- Customer Ratings
- Brand-wise Performance
- Day-wise Sales Analysis
- Interactive Filters (Month & Brand)

---

## Tools Used

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX

---

## Key Insights

- Compare sales across multiple mobile brands.
- Analyze monthly sales trends.
- Identify top-performing cities.
- Understand customer payment preferences.
- Evaluate customer ratings.
- Track daily sales performance.
